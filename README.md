# XFeat (vendored)

This directory is a vendored copy of **XFeat — Accelerated Features for Lightweight Image Matching** (Potje et al., CVPR 2024), used by `process/localization_testing/` and the live camera demos in this repo.

**Upstream (full docs, notebooks, figures):** [github.com/verlab/accelerated_features](https://github.com/verlab/accelerated_features) · [Project page](https://www.verlab.dcc.ufmg.br/descriptors/xfeat_cvpr24/) · [Paper](https://openaccess.thecvf.com/content/CVPR2024/html/Potje_XFeat_Accelerated_Features_for_Lightweight_Image_Matching_CVPR_2024_paper.html)

Teaser images, Colab notebooks, and other large upstream-only assets are omitted here to keep the tree small.
